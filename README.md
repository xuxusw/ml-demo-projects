# Портфолио
Содержит учебные и хакатонные проекты.  

## Проекты

### 1. Рекомендательная система (MAI & PSB Hackathon)
**Папка:** `01-psb-recsystem-hackathon`  
**Задача:** Разработка рекомендательной системы банковских продуктов.  
**Стек:** Python, Pandas, NumPy, SQL, Clickhouse.  
**Особенности:**
- Предобработка и анализ данных клиентов.
- Построение нескольких моделей для сравнения.
- Оценка качества рекомендаций.

### 2. Фреймворк для нейронных сетей (Lab)
**Папка:** `02-nn-framework-lab`  
**Задача:** Реализация базового фреймворка для обучения нейронных сетей с нуля.  
**Стек:** Python, NumPy (без использования готовых библиотек вроде PyTorch/TensorFlow).  
**Особенности:**
- Реализация нейронов, слоев и алгоритма обратного распространения ошибки.
- Написание собственных оптимизаторов и функций активации.
- Демонстрация работы на простых датасетах.
